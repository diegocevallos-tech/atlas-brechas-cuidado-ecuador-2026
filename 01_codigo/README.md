# Codigo

Pipeline Python completo y reproducible para el calculo del IBTC y generacion de mapas.

| Archivo | Descripcion |
|---------|-------------|
| config.py | Parametros editables centralizados |
| 01_pipeline_ibtc.py | ETL + calculo IBTC completo |
| 02_generar_mapas.py | Mapas estaticos e interactivos |
| requirements.txt | Dependencias Python |

Para ejecutar: pip install -r requirements.txt && python 01_pipeline_ibtc.py
